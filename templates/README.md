
# Protocol Core Templates

This folder contains the reusable core components of the Modular Multi-State AI Teaching Protocol (MMA-TP).

MMA-TP is not a model, plugin, API wrapper, or fine-tuning method. It is an interaction-level protocol for structuring how a capable LLM behaves during multi-turn tutoring sessions.

The protocol works by pairing two components:


Runtime Prompt
+
Structured JSON Specification


Then simply copying and pasting them into any LLM (Chat GPT & Claude Preferred)
Free usage limits are sufficient, no premium plans or sign in is necessary for either.


If you are an educator and not comfortable writing in the JSON format, the different templates do not have to be filled out manually, and can be re-written by any major LLM.
Just ensure to familiarize yourself with how each template differs, and which parameters to keep in mind (check README file connected to each template folder). An LLM will fill in the blanks where you do not, so be sure to create responsibly if you are planning to share.
